# Amazon Prime Dashboard with Tableau

This repository contains a Tableau dashboard that analyzes Amazon Prime show data from a dataset spanning from 1920 to 2021. The dashboard includes several key performance indicators (KPIs) and visualizations, such as show types, the total number of shows by country, the top 10 shows by rating, shows by release year, and the top 10 genres.
